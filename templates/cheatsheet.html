💬 How to Talk to Your AI

Just type naturally No special commands needed
Ask questions "What is...?", "How do I...?"
Paste code It can read and analyze code
Share links It can read web pages

🔧 Useful Commands

Explain like I'm 5 Simple explanations
Help me with... Get assistance with a task
What if... Explore ideas
Compare X and Y Get comparisons
Summarize this... Quick summaries
💡 Pro Tips:
• Be specific with your questions for better answers
• If the answer isn't right, ask again differently
• You can ask it to re-explain if you don't understand

⚠️ What to Avoid

Sharing passwords Never share login info
Personal secrets Keep sensitive info offline
Sensitive data Don't paste private documents
