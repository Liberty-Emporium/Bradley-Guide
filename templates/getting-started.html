📋 Getting Started

Follow these steps to set up your AI assistant

📋 Getting Started ✨ What It Can Do 📝 Cheat Sheet ❓ FAQ
1

Go to Kilo.ai

Open your web browser and go to kilo.ai

🔗 Visit kilo.ai
📸 Screenshot: Kilo.ai homepage
2

Create an Account

Click the sign-up button and create your account. You can use:

• Email — regular email and password
• GitHub — if you already have a GitHub account
• Google — sign in with your Google account

📸 Screenshot: Sign up form
3

Start an OpenClaw Instance

Once logged in, look for the option to start a new OpenClaw instance. This is your personal AI assistant!

Follow the on-screen instructions to get it set up. The setup wizard will help you connect it to:

• Telegram — message the AI like texting a friend
• WhatsApp — if you prefer WhatsApp
• Terminal — use it from command line

📸 Screenshot: OpenClaw setup screen
4

Connect to Telegram (Optional)

If you want to message your AI through Telegram:

1. Go to Telegram and search for your bot (the setup will show you how)
2. Start a conversation with it
3. Send a message and see the response!

💡 Pro Tip: You can talk to your AI from anywhere using Telegram. It feels like texting a smart friend!
5

Start Using It!

That's it! You're ready to go. Try asking your AI something like:

• "Help me write a Python program"
• "What is the weather today?"
• "Explain how computers work"
• "Help me write an email"

🎉 Congratulations! You now have your own AI assistant. Have fun exploring everything it can do!
