# Bradley's AI Guide - Project Specification

## Purpose
A step-by-step tutorial website to teach Bradley how to use AI assistants (OpenClaw/Kilo) for daily computer tasks.

## Target User
Bradley — someone new to AI assistants, needs clear visual instructions.

## Content Sections

### 1. Welcome
- What is OpenClaw?
- What can it do for you?
- Why use AI for daily tasks?

### 2. Getting Started
- Step 1: Go to kilo.ai
- Step 2: Create an account
- Step 3: Start a new OpenClaw instance
- Step 4: Connect to Telegram (optional)
- Screenshots for each step

### 3. Daily Activities
What Bradley can use it for:
- 💻 Programming help
- 📧 Email assistance  
- 🔍 Research & web searches
- 📅 Calendar management
- 📝 Writing & editing
- 💡 Problem solving
- 🛠️ Tech troubleshooting

### 4. Commands Cheat Sheet
Quick reference for common commands

### 5. FAQ
- Is it free?
- Is it safe?
- Can it do X?

---

## Technical
- Simple Flask app
- Clean, friendly UI
- Mobile-responsive
- Step-by-step format with images
